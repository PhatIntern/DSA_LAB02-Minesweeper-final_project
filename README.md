# Minesweeper Game (Java)

A Minesweeper game developed using Java and Java Swing.

This project implements the classic Minesweeper gameplay with graphical user interface, recursive DFS reveal algorithm, flagging system, difficulty selection, and restart functionality.

---

## Features

- Java Swing GUI
- Recursive DFS reveal algorithm
- Flagging system
- Difficulty selection
- First click always safe
- Bomb reveal on lose
- Restart game option
- Colored numbered cells
- Random mine generation

---

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)
- DFS / Recursion
- Git & GitHub
